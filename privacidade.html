Politica de Privacidade

Ultima atualizacao: 13 de abril de 2026

1. Introducao

O NetEscol valoriza a privacidade dos seus usuarios. Esta Politica descreve como coletamos, usamos, armazenamos e protegemos as informacoes pessoais tratadas pela nossa Plataforma, em conformidade com a Lei Geral de Protecao de Dados (LGPD - Lei 13.709/2018).

2. Dados que Coletamos

Os dados tratados pela Plataforma incluem:

  • Dados de alunos: nome, data de nascimento, CPF, endereco, filiacao, dados de saude (quando necessario), foto.
  • Dados de responsaveis: nome, CPF, telefone, e-mail, endereco, grau de parentesco.
  • Dados de servidores: nome, CPF, e-mail, telefone, cargo, formacao, dados funcionais.
  • Dados de acesso: e-mail, senha (criptografada), endereco IP, data/hora de acesso.
  • Dados de transporte: localizacao GPS dos veiculos durante o horario de operacao.

3. Finalidade do Tratamento

Os dados sao tratados exclusivamente para:

  • Gestao academica (matriculas, notas, frequencia, boletins).
  • Gestao do transporte escolar (rotas, GPS, embarque/desembarque).
  • Gestao administrativa (RH, financeiro, merenda, patrimonio).
  • Comunicacao entre escola, pais e secretaria de educacao.
  • Geracao de relatorios e prestacao de contas (transparencia publica).
  • Cumprimento de obrigacoes legais (EDUCACENSO, FNDE, etc.).

4. Base Legal

O tratamento de dados e realizado com base em:

  • Execucao de contrato: prestacao do servico contratado pela prefeitura.
  • Obrigacao legal: cumprimento de normas educacionais (LDB, EDUCACENSO).
  • Interesse publico: gestao de politicas publicas de educacao.
  • Consentimento: quando aplicavel (uso de imagem, notificacoes push).

5. Compartilhamento de Dados

Os dados NAO sao vendidos ou compartilhados com terceiros para fins comerciais. O compartilhamento ocorre apenas:

  • Com a prefeitura municipal contratante (controladora dos dados).
  • Com orgaos governamentais quando exigido por lei (INEP, FNDE, TCE).
  • Com prestadores de servico essenciais (hospedagem, e-mail) sob contratos de confidencialidade.

6. Armazenamento e Seguranca

  • Dados armazenados em servidores seguros com criptografia em transito (HTTPS/TLS).
  • Senhas criptografadas com hash bcrypt (irreversivel).
  • Autenticacao via JWT com tokens httpOnly.
  • Backup automatico diario em nuvem (OneDrive).
  • Isolamento de dados entre municipios (arquitetura multi-tenant).
  • Monitoramento de erros via Sentry.
  • Auditoria de acoes dos usuarios (logs de atividade).

7. Retencao de Dados

Os dados sao mantidos enquanto o contrato estiver vigente. Apos o encerramento:

  • Dados sao mantidos por 90 dias para exportacao.
  • Apos 90 dias, sao anonimizados ou excluidos permanentemente.
  • Dados necessarios para obrigacoes legais podem ser retidos pelo prazo legal.

8. Direitos dos Titulares

Conforme a LGPD, os titulares dos dados tem direito a:

  • Confirmacao e acesso aos dados pessoais.
  • Correcao de dados incompletos ou desatualizados.
  • Anonimizacao, bloqueio ou eliminacao de dados desnecessarios.
  • Portabilidade dos dados.
  • Informacao sobre compartilhamento com terceiros.
  • Revogacao do consentimento.

Para exercer esses direitos, entre em contato pelo e-mail privacidade@netescol.com.br.

9. Cookies

A Plataforma utiliza cookies essenciais para autenticacao e funcionamento do sistema. Nao utilizamos cookies de rastreamento ou publicidade.
