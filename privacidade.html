Política de Privacidade

Última atualização: 29 de abril de 2026

1. Introdução

O NetEscol valoriza a privacidade dos seus usuários. Esta Política descreve como coletamos, usamos, armazenamos e protegemos as informações pessoais tratadas pela nossa Plataforma, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

2. Dados que Coletamos

Os dados tratados pela Plataforma incluem:

  • Dados de alunos: nome, data de nascimento, CPF, endereço, filiação, dados de saúde (quando necessário), foto.
  • Dados de responsáveis: nome, CPF, telefone, e-mail, endereço, grau de parentesco.
  • Dados de servidores: nome, CPF, e-mail, telefone, cargo, formação, dados funcionais.
  • Dados de acesso: e-mail, senha (criptografada), endereço IP, data/hora de acesso.
  • Dados de transporte: localização GPS dos veículos durante o horário de operação.

3. Finalidade do Tratamento

Os dados são tratados exclusivamente para:

  • Gestão acadêmica (matrículas, notas, frequência, boletins).
  • Gestão do transporte escolar (rotas, GPS, embarque/desembarque).
  • Gestão administrativa (RH, financeiro, merenda, patrimônio).
  • Comunicação entre escola, pais e secretaria de educação.
  • Geração de relatórios e prestação de contas (transparência pública).
  • Cumprimento de obrigações legais (EDUCACENSO, FNDE, etc.).

4. Base Legal

O tratamento de dados é realizado com base em:

  • Execução de contrato: prestação do serviço contratado pela prefeitura.
  • Obrigação legal: cumprimento de normas educacionais (LDB, EDUCACENSO).
  • Interesse público: gestão de políticas públicas de educação.
  • Consentimento: quando aplicável (uso de imagem, notificações push).

5. Compartilhamento de Dados

Os dados NÃO são vendidos ou compartilhados com terceiros para fins comerciais. O compartilhamento ocorre apenas:

  • Com a prefeitura municipal contratante (controladora dos dados).
  • Com órgãos governamentais quando exigido por lei (INEP, FNDE, TCE).
  • Com prestadores de serviço essenciais (hospedagem, e-mail) sob contratos de confidencialidade.

6. Armazenamento e Segurança

  • Dados armazenados em servidores seguros com criptografia em trânsito (HTTPS/TLS).
  • Senhas criptografadas com hash bcrypt (irreversível).
  • Autenticação via JWT com tokens httpOnly.
  • Backup automático diário em nuvem (OneDrive).
  • Isolamento de dados entre municípios (arquitetura multi-tenant).
  • Monitoramento de erros via Sentry.
  • Auditoria de ações dos usuários (logs de atividade).

7. Retenção de Dados

Os dados são mantidos enquanto o contrato estiver vigente. Após o encerramento:

  • Dados são mantidos por 90 dias para exportação.
  • Após 90 dias, são anonimizados ou excluídos permanentemente.
  • Dados necessários para obrigações legais podem ser retidos pelo prazo legal.

8. Direitos dos Titulares

Conforme a LGPD, os titulares dos dados têm direito a:

  • Confirmação e acesso aos dados pessoais.
  • Correção de dados incompletos ou desatualizados.
  • Anonimização, bloqueio ou eliminação de dados desnecessários.
  • Portabilidade dos dados.
  • Informação sobre compartilhamento com terceiros.
  • Revogação do consentimento.

Para exercer esses direitos, entre em contato pelo e-mail privacidade@netescol.com.br.

9. Cookies

A Plataforma utiliza cookies essenciais para autenticação e funcionamento do sistema. Não utilizamos cookies de rastreamento ou publicidade.
